# Citation Format Reference

Full patterns for APA 7th Edition and MLA 9th Edition across all common source types.

---

## APA 7th Edition

### Journal Articles

**Standard article**
```
Author, A. A., & Author, B. B. (Year). Title of article in sentence case. *Journal Name in Title Case*, *Volume*(Issue), start–end. https://doi.org/xxxxx
```

**Article with DOI (no URL needed)**
```
Grady, J. S., Her, M., Moreno, G., Perez, C., & Yelinek, J. (2019). Emotions in storybooks: A comparison of storybooks that represent ethnic and racial groups in the United States. *Psychology of Popular Media Culture*, *8*(3), 207–217. https://doi.org/10.1037/ppm0000185
```

**Article without DOI (with URL)**
```
Stuster, J. (2010). Behavioral issues associated with long-duration space expeditions: Review and analysis of astronaut journals. *Aviation, Space, and Environmental Medicine*, *81*(11), 1033–1038. https://www.ingentaconnect.com/content/asma/asem
```

**Advance online publication**
```
Reysen, S., & Lloyd, J. D. (2020). Fan identification. *Current Opinion in Psychology*. Advance online publication. https://doi.org/10.1016/j.copsyc.2020.06.007
```

---

### Books

**Entire book**
```
Author, A. A. (Year). *Title of work: Capital letter also for subtitle*. Publisher. https://doi.org/xxxxx
```

**Edited book**
```
Editor, E. E. (Ed.). (Year). *Title of work: Capital letter also for subtitle*. Publisher.
```

**Chapter in edited book**
```
Author, A. A. (Year). Title of chapter. In E. E. Editor (Ed.), *Title of work: Capital letter also for subtitle* (pp. pages). Publisher. https://doi.org/xxxxx
```

**Edition other than first**
```
Author, A. A. (Year). *Title of work* (Xth ed.). Publisher.
```

---

### Websites & Online Sources

**Webpage with individual author**
```
Author, A. A. (Year, Month Day). Title of page. *Site Name*. https://www.example.com/path
```

**Webpage with organization as author**
```
Organization Name. (Year, Month Day). Title of page. https://www.example.com/path
```

**No date**
```
Author, A. A. (n.d.). Title of page. *Site Name*. https://www.example.com/path
```

**Retrieved date (use only if content changes frequently)**
```
Author, A. A. (Year). Title of page. *Site Name*. Retrieved Month Day, Year, from https://example.com
```

---

### Reports & Government Documents

**Government report**
```
National Cancer Institute. (2019). *Taking time: Support for people with cancer* (NIH Publication No. 18-2059). U.S. Department of Health and Human Services, National Institutes of Health. https://www.cancer.gov/publications/patient-education/taking-time
```

**Corporate/think tank report**
```
Author, A. A. (Year). *Title of report* (Report No. xxx). Organization. https://doi.org/xxxxx
```

---

### Conference Papers

**Published in proceedings**
```
Author, A. A. (Year). Title of paper. In E. E. Editor (Ed.), *Title of proceedings* (pp. pages). Publisher. https://doi.org/xxxxx
```

**Unpublished conference paper**
```
Author, A. A. (Year, Month). *Title of paper* [Paper presentation]. Conference Name, City, Country.
```

---

### Dissertations & Theses

**ProQuest/institutional repository**
```
Author, A. A. (Year). *Title of dissertation* [Doctoral dissertation, University Name]. ProQuest Dissertations and Theses. URL
```

---

### In-Text Citations (APA)

| Situation | Format |
|-----------|--------|
| One author | (Smith, 2020) |
| Two authors | (Smith & Jones, 2020) |
| Three or more | (Smith et al., 2020) |
| Direct quote | (Smith, 2020, p. 45) |
| Narrative | Smith (2020) found that… |
| No author | ("Title Fragment," Year) |
| No date | (Smith, n.d.) |

## MLA 9th Edition

### Core Container Structure

```
Author Last, First. "Source Title." *Container Title*, Other Contributors, Version, Number, Publisher, Date, Location.
```

Not all fields apply to every source — include only what is relevant.

---

### Journal Articles

**Print**
```
Author Last, First. "Article Title." *Journal Name*, vol. #, no. #, Year, pp. #–#.
```

**Online with DOI**
```
Author Last, First. "Article Title." *Journal Name*, vol. #, no. #, Year, pp. #–#. https://doi.org/xxxxx.
```

**Online without DOI**
```
Author Last, First. "Article Title." *Journal Name*, vol. #, no. #, Year, pp. #–#. *Database Name*, URL.
```

---

### Books

**Single author**
```
Author Last, First. *Book Title: Subtitle*. Publisher, Year.
```

**Multiple authors**
```
First Author Last, First, and Second Author First Last. *Book Title*. Publisher, Year.
```

**Three or more authors**
```
First Author Last, First, et al. *Book Title*. Publisher, Year.
```

**Edited book**
```
Editor Last, First, editor. *Book Title*. Publisher, Year.
```

**Chapter in edited book**
```
Author Last, First. "Chapter Title." *Book Title*, edited by Editor First Last, Publisher, Year, pp. #–#.
```

**Edition other than first**
```
Author Last, First. *Book Title*. Xth ed., Publisher, Year.
```

---

### Websites & Online Sources

**With author**
```
Author Last, First. "Page Title." *Site Name*, Day Mon. Year, URL.
```

**Organization as author**
```
Organization Name. "Page Title." *Site Name*, Day Mon. Year, URL.
```

**No date**
```
Author Last, First. "Page Title." *Site Name*, URL. Accessed Day Mon. Year.
```

---

### Reports & Government Documents

```
Organization Name. *Report Title*. Publisher, Year. URL.
```

---

### In-Text Citations (MLA)

| Situation | Format |
|-----------|--------|
| One author | (Smith 45) |
| Two authors | (Smith and Jones 45) |
| Three or more | (Smith et al. 45) |
| No page (website) | (Smith) |
| No author | ("Article Title" 45) |
| Same author, multiple works | (Smith, *Title* 45) |
| Narrative | Smith argues that… (45). |

**MLA does NOT include year in parenthetical citation.**

---

## Quick Comparison: APA vs MLA

| Feature | APA 7th | MLA 9th |
|---------|---------|---------|
| In-text includes | Author, Year | Author, Page# |
| Reference list title | References | Works Cited |
| Journal title | *Italicized* | *Italicized* |
| Article title | Sentence case, no quotes | "Title Case in Quotes" |
| Book title | *Italics*, Sentence case | *Italics*, Title Case |
| DOI format | https://doi.org/xxx | https://doi.org/xxx |
| Publisher in journal? | No | No |
| Publisher in books? | Yes | Yes |
| Date position | After author | Near end of entry |
