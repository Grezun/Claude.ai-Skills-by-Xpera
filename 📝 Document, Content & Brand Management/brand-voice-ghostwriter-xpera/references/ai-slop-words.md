# AI Slop — Banned Words Reference

These words signal AI-generated filler. Ban them all. When you spot one, rewrite the sentence using plain English.

---

## The A-List (Most Recognizable Offenders)

| Banned | Use Instead |
|--------|-------------|
| delve (into) | look at, explore, dig into |
| testament (to) | proof of, sign of |
| revolutionize | change, improve, rethink |
| leverage (verb) | use, apply, build on |
| seamlessly | just say it works — or describe how |
| game-changing | say what actually changed |
| transformative | say what transformed |
| unprecedented | new, first, never before — or cut it |
| cutting-edge | new, latest, current |
| state-of-the-art | see above |
| best-in-class | say the specific advantage |
| world-class | say the specific advantage |
| groundbreaking | say what ground it broke |
| robust | strong, reliable, flexible — pick one |
| holistic | complete, full, end-to-end |
| synergy | working together, shared goals |
| empower | let, give, help, enable |
| harness | use, tap into, build on |
| foster | build, grow, support |
| spearhead | lead, run, start |
| navigate (complexity) | deal with, work through, handle |
| elevate | improve, raise, lift |
| unlock (potential) | just say what they'll be able to do |

## Adjective Bloat (Overused Intensifiers)

Ban these when they're just decoration:

- innovative / innovation (overused — say what's new)
- comprehensive (overused — say what's included)
- dynamic (meaningless — cut it)
- scalable (only use when scale is the actual point)
- strategic (only use when the strategy is the point)
- seamless (usually signals the thing isn't actually seamless)
- intuitive (show it, don't claim it)
- powerful (show it, don't claim it)
- best (needs proof)
- leading (needs proof)
- trusted (needs proof)

## Business Clichés (Human Slop)

These aren't AI-specific — they're what humans write when they're not thinking. Same result: vague, forgettable copy.

| Cliché | Fix |
|--------|-----|
| pulling ahead | outperforming, growing faster, leading — or say the actual metric |
| runs on rails | say what actually runs smoothly and why |
| compounding (as vague intensifier) | say what's growing and by how much |
| stay ahead of the curve | say ahead of what, specifically |
| move fast | say how fast and toward what |
| level up | improve, get better at, advance |
| at scale | say the actual scale |
| where the rubber meets the road | cut it |
| low-hanging fruit | easy wins, quick fixes — then list them |
| boiling the ocean | trying to do too much — say what specifically |
| Every [audience] knows the feeling… | Start with the actual observation |

## Opener Red Flags (First Line Killers)

If your draft starts with any of these, delete and rewrite:

- "In today's…"
- "In an era where…"
- "As [X] continues to…"
- "It's no secret that…"
- "We're thrilled to announce…" (use "Today we're launching…" or just state what launched)
- "I'm excited to share…"
- "[Company] is proud to…"
- "Every [audience] knows the feeling…" (start with the actual observation)
- "In this guide/post/article, we'll…" (just start doing it — don't announce that you will)

---

## Closer Red Flags (Last Line Killers)

- "In conclusion…"
- "In summary…"
- "To sum up…"
- "As you can see…"
- "We hope this was helpful."
- "Feel free to reach out."
- "Starting with the question most people never think to ask…" (cliffhanger tease — say the thing)
- Any sentence that promises what the next section will cover without delivering it

End with the actual takeaway, not a label for it. A cliffhanger is not a takeaway.
