# Brand Voice Ghostwriter

## Overview

A fillable brand voice framework. Organizations drop in their specific details; universal rules (AI slop ban, conversational tone, H2/H3 structure) apply regardless of brand.

**Core principle:** Write like a sharp human, not a press release. Say what you mean. Cut the rest.

## Step 2 — Universal Rules (Always Apply)

### Tone
- Conversational. Write like explaining to a smart colleague, not presenting to a board.
- Active voice. Short sentences. Real words.
- State the opinion. Don't hedge into mush.
- If a sentence can be cut without losing meaning, cut it.

### Structure
- **H2** for main sections, **H3** for subsections.
- Sentence case for all headings — not Title Case.
- Lead with the point. No wind-up.
- Paragraphs: 2–3 sentences for web and social. 4 max.
- No "In today's world…" openers.
- No "In this guide/post/article, we'll…" transitions — just do it.
- No "In conclusion…" closers. End with the takeaway.
- No cliffhanger endings ("Starting with the question no one asks…"). That's a tease, not a takeaway. Say the thing.

### AI Slop — Banned Always
See `references/ai-slop-words.md` for the full list.

**Core offenders to memorize:**
`delve` · `testament` · `revolutionize` · `leverage` (as verb) · `seamlessly` · `game-changing` · `transformative` · `holistic` · `robust` · `synergy` · `cutting-edge` · `unprecedented` · `empower` · `harness` · `navigate complexity` · `foster` · `spearhead` · `it's worth noting` · `it goes without saying`

**Rule:** If you catch yourself writing any of these — stop. Rewrite the sentence from scratch using plain words.

## Step 3 — Content-Type Formats

### Blog Post
- **Hook:** 1–2 sentences. Start with the problem or a sharp observation. No rhetorical questions.
- **Body:** H2 sections, 150–300 words each. No filler between sections.
- **Close:** One actionable takeaway. Not a recap.

### Launch Announcement
- **Lead:** What changed + why it matters — in the first sentence.
- **Body:** Who it's for, what it does, what's different. Specifics over adjectives.
- **CTA:** One clear next step.

### LinkedIn Post
- First line does the work — write it to stand alone.
- No hook bait ("I have a hot take…", "Unpopular opinion:").
- 3–5 short paragraphs or bullets.
- Max 2 hashtags, or none.

### Product Copy
- Benefit first, feature second.
- Avoid "solution" and "platform" as standalone nouns.
- Short. Punchy. Present tense.

### Email
- Subject line: specific and human — not "Exciting Update!" or "Following up…"
- Body: what you need + why, in 3 sentences or fewer.
- One CTA per email.

### Case Study
- Structure: **Problem → Approach → Result**
- Lead with the outcome. Don't bury the number.
- Quote the person, not the company.
- Use past tense for what happened; present tense for what's true now.

## Quick Reference — Common Mistakes

| Mistake | Fix |
|---------|-----|
| "Leverage our robust platform" | "Use our platform" |
| "In today's fast-paced world…" | Start with the actual point |
| "Was implemented by the team" | "The team built it" |
| "It's worth noting that…" | Just say it |
| "Seamlessly integrates" | "Works with X out of the box" |
| "Drive transformative outcomes" | Name the actual outcome |
| "In conclusion…" | End with the takeaway, no label |
| Three-sentence intro before the point | Cut to the first sentence that matters |
| "In this guide/post, we'll cover…" | Just start covering it |
| Ending with a cliffhanger tease | Say the actual thing, don't tease it |
| "Every [audience] knows the feeling…" | Start with the actual observation |
